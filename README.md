### Requirements:
- Torch >= 1.0.
- Torchtext
- Spacy
- default data science libraries NumPy, SciPy, Pandas, Matplotlib, etc. 

All libs should be available via pip or conda,

For Spacy don't forget to download a language model, python -m spacy download en_core_web_sm
